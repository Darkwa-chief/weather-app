About WeatherApp

WeatherApp is a modern, full-featured static UI weather application.

This multi-page design shows:

  • Current weather details with animated icons
  • 7-day forecast carousel
  • Weather alerts panel
  • Recent cities panel
  • Full-screen, responsive design

JavaScript/API integration will be added later to make it fully functional.
